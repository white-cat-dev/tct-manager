## Установка

### Зависимости
```composer install```

### Миграции
```php artisan migrate```

### Начальные данные
```php artisan db:seed```

### Импорт продуктов
```php artisan import:products```

### Импорт материалов
```php artisan import:materials```

### Импорт рецептов
```php artisan import:recipes```

### Создание пользователя
```php artisan make:user```
